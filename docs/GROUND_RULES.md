# Mega Launcher — Ground Rules

모든 프로젝트와 태스크 관리에 적용되는 규칙.

---

## 1. 진행률은 태스크 체크 수로만 계산한다

- 진행률(%) = 완료된 태스크 수 / 전체 태스크 수 × 100
- 태스크가 하나도 체크되지 않으면 0%
- 모든 태스크가 체크되면 100% → 프로젝트 상태 자동으로 "완료"로 전환
- 태스크 체크 없이 진행률이 올라가서는 안 된다

## 2. 프로젝트 생성 시 태스크를 반드시 채운다

- 프로젝트가 생성될 때, 완료까지 필요한 태스크를 예상해서 미리 등록한다
- 완벽하지 않아도 되며, 간단하게만 작성한다
- 태스크 0개인 프로젝트는 허용하지 않는다

## 3. 진행률 직접 수정 불가

- 진행률 필드를 수동으로 입력하거나 수정하는 UI/API를 제공하지 않는다
- 진행률은 항상 태스크 상태에서 자동 계산된다

---

## 구현 체크리스트

- [ ] 태스크 완료 시 진행률 자동 재계산
- [ ] 모든 태스크 완료 시 프로젝트 상태 → "완료" 자동 전환
- [ ] 프로젝트 생성 폼에서 태스크 최소 1개 필수 입력
- [ ] 진행률 수동 입력 필드 제거
